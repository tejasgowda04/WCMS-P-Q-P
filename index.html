1a
Describe the different types of Content Management System (CMS).
5 MarksPart A
▾
A Content Management System (CMS) is software that helps create, manage, and modify digital content on a website without needing technical coding knowledge.

Types of CMS:

  • 1
    Web CMS (WCMS) – Manages web content on websites. Example: WordPress, Joomla, Drupal
  • 2
    Enterprise CMS (ECMS) – Used by large organizations to manage documents and records. Example: Microsoft SharePoint
  • 3
    Document Management System (DMS) – Manages documents, version control, and file storage
  • 4
    Digital Asset Management (DAM) – Manages media assets like images, videos, audio files
  • 5
    Component CMS (CCMS) – Manages content at the component/module level for reuse
  • 6
    Learning Management System (LMS) – Manages educational content and online courses. Example: Moodle
  • 7
    Mobile CMS – Manages content specifically for mobile apps and responsive sites
Remember: WordPress = most popular Web CMS | Moodle = LMS | SharePoint = Enterprise CMS

Explain the different types of Hosting.
5 Marks
▾
Web Hosting is a service that provides storage space and internet connectivity to make websites accessible online.
  • 1
    Shared Hosting – Multiple websites share one server's resources. Cheapest option. Suitable for small websites. Limited performance.
  • 2
    VPS (Virtual Private Server) – Physical server divided into virtual servers. Each site gets dedicated resources. More control, moderate cost.
  • 3

  • 5
    Managed Hosting – Provider handles all technical tasks like updates, security, backups
  • 6
    Reseller Hosting – Purchase bulk hosting and resell to others; popular for web agencies
  • 7
    Colocation Hosting – Own your server hardware but keep it at provider's data center

  • 2
    Box Model – Every element has: Content → Padding → Border → Margin
  • 3
    Layout – Flexbox (1D layout), Grid (2D layout) for page structure
  • 4
    Typography – font-family, font-size, font-weight, line-height, text-align
  • 5
    Colors & Backgrounds – background-color, color, gradients, background-image
  • 6
    Responsive Design – Media queries (@media) to adapt layout for different screen sizes
  • 7
    Animations – transition and @keyframes for smooth effects
  • 8
    CSS Variables – :root { --color: #fff } for reusable design tokens

CSS Frameworks

Bootstrap, Tailwind CSS, Foundation – pre-built components

3 Ways to Use CSS

Inline (style=""), Internal (<style>), External (.css file)

Sound Sync – Add voice-over, music, and sound effects

What are the different formats available for E-Publications?
5 Marks
▾
  • 1
    PDF (Portable Document Format) – Fixed layout, preserves formatting, universally supported, easy to print. Best for reports and textbooks.
  • 2
    EPUB (Electronic Publication) – Reflowable text, adjusts to screen size, standard format for most e-readers. Open standard by IDPF.
  • 3
    MOBI – Amazon Kindle's original format. Limited to Kindle devices and apps.
  • 4
    AZW3 (Kindle Format 8) – Advanced Amazon format with better CSS support, replacing MOBI.
  • 5
    HTML – Web-based, fully interactive, supports multimedia. Accessed via browsers.
  • 6
    TXT – Plain text, smallest file size, no formatting. Used for simple documents.
  • 7
    DOCX – Editable Microsoft Word format. Often used before converting to other formats.
